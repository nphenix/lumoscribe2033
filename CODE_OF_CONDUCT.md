# lumoscribe2033 项目行为准则

## 我们的承诺

我们致力于为 lumoscribe2033 项目创造一个友好、开放和欢迎的环境，无论您的性别认同、性取向、残障程度、体能状况、外貌、体型、种族、民族、年龄、宗教信仰或技术经验如何。

我们承诺为所有贡献者和社区成员提供一个无骚扰的体验。我们不容忍任何形式的骚扰、歧视或排斥行为。

## 我们的职责

项目维护者有责任澄清本行为准则的规范，并且会以实际行为作为判断标准，采取适当的行动来纠正问题。

项目维护者有义务对违反本行为准则的任何行为做出快速且公正的回应。

## 涵盖范围

本行为准则适用于所有项目空间，以及在公开场合代表本项目或其社区时的个人行为。

项目空间包括：

- 官方仓库和其评论区
- 官方文档和其评论区
- 官方社交媒体账号
- 官方邮件列表、讨论组和聊天室
- 官方活动、会议和聚会
- 任何由项目维护者组织或赞助的活动

## 行为规范

### 我们鼓励的行为：

- 使用友好和包容性的语言
- 尊重不同的观点和经验
- 以建设性的方式接受批评
- 关注他人的福祉
- 对其他社区成员表示同理心

### 我们不接受的行为：

- 使用带有攻击性、排他性或不受欢迎的语言
- 恶意对待、骚扰或霸凌他人
- 发表不当的性言论或性暗示
- 公开或私下威胁或恐吓他人
- 未经同意发布他人的私人信息（如真实姓名、住址等）
- 其他不专业的或破坏性行为

## 执行规范

如果某人表现出我们不接受的行为，项目维护者有权采取他们认为合适的行动，包括但不限于：

- 对不当行为发出警告
- 删除不当的评论或贡献
- 暂时或永久禁止违规者参与项目（包括贡献和使用）
- 将严重违规行为报告给相关机构

## 对维护者的期望

项目维护者应该：

- 成为包容性社区的榜样
- 采纳协作和建设性的解决方案
- 准确并公正地执行本行为准则
- 对违规行为进行保密处理，除非需要公开透明

## 致谢

本行为准则基于 [Contributor Covenant](https://www.contributor-covenant.org) 版本 2.1，这是开源社区广泛采用的标准行为准则。

## 报告机制

如果您发现任何违反本行为准则的行为，或者有任何相关问题，请通过以下方式联系我们：

- 发送邮件至：18210768480@139.com
- 提交 GitHub Issue 并标记为 `code-of-conduct`

对于维护者违反行为准则的情况，您可以联系项目所有者或通过上述渠道报告。

## 报告处理流程

1. **接收报告**：我们会在收到报告后 48 小时内确认收悉
2. **调查评估**：项目维护者将对事件进行调查和评估
3. **采取行动**：根据调查结果采取适当的纠正措施
4. **反馈结果**：在保护隐私的前提下，向报告者反馈处理结果
