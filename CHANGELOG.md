# 更新日志

本文档记录 lumoscribe2033 的重要变更。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [0.1.0] - 2025-11-19

### 🚀 技术栈预览版发布

lumoscribe2033 技术栈预览版正式发布！本版本展示了项目的基础架构和技术能力，为后续功能开发奠定坚实基础。

### ✨ 新增功能

#### 🏗️ 基础架构
- **项目结构**: 完整的分层架构设计和模块化组织
- **配置管理**: 基于 Pydantic 的环境配置系统
- **依赖管理**: 现代化的 Python 项目依赖管理
- **文档体系**: 全面的中文文档和使用指南

#### 🤖 AI 与 RAG 能力
- **LangChain 1.0**: 集成 LangChain 1.0.6，支持代理创建和 LLM 编排
- **LlamaIndex**: 集成 LlamaIndex 0.14.8，提供 RAG 解决方案
- **多模型支持**: 支持 OpenAI 兼容 API 和本地 Ollama
- **智能路由**: 实现动态模型选择和负载均衡机制

#### 🌐 API 与服务
- **FastAPI**: 基于 FastAPI 0.121.2 的高性能异步 Web 服务
- **FastMCP**: 集成 FastMCP 2.13.0.2，支持模型上下文协议
- **RESTful API**: 完整的接口设计和自动生成文档
- **异步任务**: 基于 Arq 0.26.3 的异步任务处理系统

#### 🛠️ 开发工具链
- **CLI 工具**: 基于 Typer 0.20.0 的命令行界面
- **质量检查**: 集成 Ruff 0.14.5 和 MyPy 1.7.0 代码质量保障
- **测试体系**: 完整的单元测试、集成测试和契约测试
- **CI/CD**: GitHub Actions 自动化流程

#### 📊 可观测性
- **监控指标**: OpenTelemetry 分布式追踪和指标收集
- **日志系统**: Loguru + Structlog 结构化日志系统
- **性能分析**: 请求监控和性能指标收集
- **健康检查**: 完整的服务状态监控

#### 🗄️ 数据存储
- **关系数据库**: SQLModel + SQLite 数据库抽象
- **向量数据库**: ChromaDB 向量存储
- **图数据库**: NetworkX 图分析
- **存储抽象**: 统一的数据访问接口
